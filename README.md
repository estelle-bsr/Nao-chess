# Nao-chess
Lors de mon S5, j’ai participé à un projet en équipe de 10, divisée en trois sous-équipes : IA (décisions et détection de triche), Vision (reconnaissance des pions et de l’échiquier), et Mouvement/Son (mouvements et interactions). Le robot Nao 6 a été programmé en Python 2.7, avec gestion via Git et méthode Scrum.
